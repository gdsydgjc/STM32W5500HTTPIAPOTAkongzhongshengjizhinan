### 为什么选择HTTP通信协议？

选择HTTP通信协议的主要原因在于其协议头中包含的body长度字段。这一特性使得在OTA升级过程中可以采用断点续传的方式，这对于确保升级过程的稳定性和可靠性至关重要。
